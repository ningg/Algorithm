## 算法

几个常见场景:

1. 斐波那契数列: `Fibo`
2. 斐波那契数列: 数字判断 `FiboContainValue`
3. 链表翻转: `ReverseList`
4. 链表:有序链表归并 `MergeSortedList`
5. 链表:排序 `SortList`
6. 链表:删除重复节点 `RemoveDuplicateNode`
7. 缓存:LRU 缓存 `LruCache`
8. 树:镜像 `MirrorTree`
9. 数组:二分查找
10. 数组:查找 top k
11. 数据: 二维有序数组, 查询目标数值 `SearchOrderedArray`
12. 树: 遍历(前序、中序、后续), 深度遍历、广度遍历、层次遍历
13. 数组：2 个有序数组的交集 `TwoSortedArrayIntersection`
